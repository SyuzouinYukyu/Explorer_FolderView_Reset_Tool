# CHANGELOG

## v1.1.0

- 「フォルダー種類自動判定を無効化する」チェックボックスを追加。
- `FolderType=NotSpecified` 設定に対応。
- 事前チェックに `FolderType` 状態確認を追加。
- ログに `FolderType` 設定結果を出力。
- バージョンを v1.1.0 へ更新。

## v1.0.0

- 初版。
- Windows 11 Explorer のフォルダ表示記憶 `Bags` / `BagMRU` のバックアップ、リセット、復元に対応。
- `BagMRU Size` の GUI 設定に対応。
- Explorer 再起動、バックアップ先を開く、UTF-8 BOM 付きログ保存に対応。
- アプリケーションアイコンとフォーム左上アイコンに対応。
- 独自ライセンスを追加。
